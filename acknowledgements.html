Open-source acknowledgements

CouchPilot is built on open technologies. Thank you to the maintainers who make a free, local-first remote possible.

The CouchPilot app is built with the following open-source components, each under its own license:

  • React Native & React — the app framework (MIT).
  • Expo — build and development tooling (MIT).
  • React Navigation — in-app navigation (MIT).
  • react-native-tcp-socket — local TCP/TLS connections to the TV (MIT).
  • react-native-udp — Wake-on-LAN over UDP (MIT).

The TV control approach is informed by the wider community that has documented local TV protocols over the years. CouchPilot uses only local, on-device connections and no cloud services.

This website is hand-written static HTML, CSS, and vanilla JavaScript — no frameworks, no trackers, no cookies.

Full, versioned license texts are included with the app. Questions? Contact us.
